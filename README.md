## ICFP Contest 2014

LambdaMan vs Ghosts.

### Contents

1. Scheme to GCC-code compiler: `src/LambdaCompiler.hs`
2. DSL for generation GHC-code and AIs for Ghosts: `src/GHC/AI.hs`

To play around:

```
$ cabal sandbox init
$ cabal install
$ ./compile-bot.sh
```
